yhstat --fields 命令中可输出的字段包括多个与作业相关的性能和状态信息，这些字段用于详细描述作业的执行情况和资源使用情况。1. AveCPU 表示平均 CPU 使用时间，单位为秒。2. AvePages 表示平均使用的页面数。3. AveRSS 表示平均常驻集大小，即实际使用的物理内存。4. AveVMSize 表示平均虚拟内存大小。5. JobID 是作业的唯一标识符。6. MaxPages 表示作业过程中最大的页面数。7. MaxPagesNode 表示每个节点上的最大页面数。8. MaxPagesTask 表示每个任务的最大页面数。9. MaxRSS 表示作业过程中最大的常驻集大小。10. MaxRSSNode 表示每个节点上的最大常驻集大小。11. MaxRSSTask 表示每个任务的最大常驻集大小。12. MaxVMSize 表示作业过程中最大的虚拟内存大小。13. MaxVMSizeNode 表示每个节点上的最大虚拟内存大小。14. MaxVMSizeTask 表示每个任务的最大虚拟内存大小。15. MinCPU 表示作业过程中最小的 CPU 使用时间。16. MinCPUNode 表示每个节点上的最小 CPU 使用时间。17. MinCPUTask 表示每个任务的最小 CPU 使用时间。18. NTasks 表示作业中的任务数量。19. SystemCPU 表示系统 CPU 时间。20. TotalCPU 表示总的 CPU 时间。这些字段可以用来分析作业的性能和资源使用情况，帮助用户优化作业配置和提高计算效率。
